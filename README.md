# Matrix / Матрицы
Программа реализации основных мат. действий над матрицами в визуальной среде Лазарус. 
1. Матрицы А и Б можно изменять в размере 
2. 3 вида действий над матрицами
3. Все предупреждения об ошибках выводятся
